🤖 通过 Qoder 安装 OpenClaw

完整教程 + 提示词，让 AI 助手帮你自动化安装

⏱️ 学习时长：20 分钟  |  ⭐ 难度等级：简单  |  🛠️ 前置要求：已安装 Qoder

1准备工作

在开始之前，请确保：

  • ✅ 已安装 Qoder（代码编辑器）
  • ✅ Qoder 已配置 AI 助手（推荐通义千问）
  • ✅ 电脑已安装 Node.js（v18+）
  • ✅ 有稳定的网络连接

2打开 Qoder 并启动 AI 助手

在 Qoder 中打开 AI 助手面板（通常快捷键 Ctrl+Shift+P 或 Cmd+Shift+P）

3发送安装提示词

复制以下完整提示词发送给 Qoder 的 AI 助手：

📋 完整提示词

请帮我安装 OpenClaw（AI 助手框架），按照以下步骤执行：

## 步骤 1: 创建项目目录
在终端执行：
```bash
mkdir openclaw-workspace
cd openclaw-workspace
```

## 步骤 2: 克隆 OpenClaw 仓库
```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
```

## 步骤 3: 安装依赖
```bash
npm install
```

## 步骤 4: 创建环境变量文件
创建 .env 文件，内容：
```
DASHSCOPE_API_KEY=请替换为你的通义千问 API Key
```

## 步骤 5: 配置渠道
创建 config.yaml 文件，内容：
```yaml
gateway:
  port: 3000
  host: 0.0.0.0

channels:
  telegram:
    - token: "请替换为你的 Telegram Bot Token"
      name: "OpenClaw Bot"
      model: "qwen-plus"
      thinking: "on"
      systemPrompt: |
        你是一位专业、友好、高效的 AI 助手。
        你擅长用简洁清晰的语言解答问题。

models:
  qwen-plus:
    provider: dashscope
    apiKey: ${DASHSCOPE_API_KEY}
    model: qwen-plus
    maxTokens: 2000
```

## 步骤 6: 启动 OpenClaw
```bash
npm start
```

## 步骤 7: 验证安装
请告诉我如何验证 OpenClaw 是否安装成功，以及下一步该做什么。

请一步步执行上述命令，并在每一步完成后告诉我结果。如果遇到错误，请详细说明如何修复。

4等待 AI 执行

Qoder 的 AI 助手会自动：

  • ✅ 创建项目目录
  • ✅ 克隆 Git 仓库
  • ✅ 安装 npm 依赖
  • ✅ 创建配置文件
  • ✅ 启动 OpenClaw
💡 提示：如果 AI 询问 API Key 或 Bot Token，请提供真实值。可以从以下地址获取：
  • 通义千问 API Key: https://dashscope.console.aliyun.com/
  • Telegram Bot Token: 在 Telegram 中搜索 @BotFather

5验证安装

安装完成后，AI 会显示类似输出：

[Gateway] Starting OpenClaw Gateway...
[Channels] Loading channel plugins...
[Telegram] Bot "OpenClaw Bot" started successfully
[Gateway] Gateway is running! Listening for messages...

看到 Gateway is running! 说明安装成功！

6测试机器人

在 Telegram 中找到你的机器人，发送：

你好

预期回复：你好！我是你的 OpenClaw 助手，有什么可以帮你的吗？

🐛 常见问题

Q1: Qoder 无法执行终端命令

解决方法：在 Qoder 设置中启用终端权限，或手动在终端执行命令。

Q2: npm install 失败

解决方法：检查 Node.js 版本（需要 v18+），或尝试 npm install --legacy-peer-deps

Q3: Gateway 启动失败

检查项：

  • 端口 3000 是否被占用
  • .env 文件是否存在
  • API Key 是否正确
