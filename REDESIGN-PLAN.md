## 🏗️ 技术架构

### 前端技术栈
```
├── 框架: HTML5 + Vanilla JavaScript (保持简单)
├── 样式: Tailwind CSS (现代化 UI)
├── 图标: Font Awesome / Lucide
├── 动画: CSS Animation + GSAP (小龙虾动画)
├── 轮播: Swiper.js
└── 图表: Chart.js (数据统计)
```

### 后端/数据存储方案

**方案 A: 纯前端 + 第三方服务** (推荐初期)
```
├── 用户认证: Auth0 / Firebase Authentication
├── 文件存储: Cloudinary / AWS S3
├── 数据库: Firebase Firestore / Supabase
└── 部署: GitHub Pages + Serverless Functions
```

**方案 B: 自建后端** (后期扩展)
```
├── 后端: Node.js + Express
├── 数据库: MongoDB / PostgreSQL
├── 文件存储: MinIO / AWS S3
└── 部署: Vercel / Railway / 阿里云
```

### 2. 动态小龙虾设计

```css
/* 小龙虾动画效果 */
@keyframes lobsterWave {
  0%, 100% { transform: rotate(0deg); }
  25% { transform: rotate(-10deg); }
  75% { transform: rotate(10deg); }
}

@keyframes lobsterWalk {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-5px); }
}

.lobster-logo {
  animation: lobsterWave 2s ease-in-out infinite;
}

.lobster-logo:hover {
  animation: lobsterWalk 0.5s ease-in-out infinite;
}
```

**SVG 小龙虾设计要点**:
- 🦞 红色主体 + 大钳子
- 👀 可爱的大眼睛
- 悬停时挥动钳子
- 点击时有反馈动画

- 🌙 **切换按钮**：右上角，平滑过渡动画
- 💾 **偏好记忆**：自动保存用户选择到 localStorage
- 🎨 **自动适配**：根据系统偏好自动切换

**热门 Banner 轮播**:
- 自动轮播，5秒切换
- 显示文章缩略图、标题、点赞数
- 点击跳转详情页
- 指示器 + 左右箭头控制

**内容分类**:
- 双维度筛选：方向（OpenClaw/Skills/部署）+ 类型（教程/案例/视频）
- 标签式 UI，支持多选

**内容卡片网格**:
- 响应式：桌面 4列 / 平板 2列 / 手机 1列
- 卡片包含：封面图、标题、摘要、阅读量、点赞数、作者、发布时间
- 悬停效果：上浮 + 阴影增强
